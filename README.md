# AI Resume Builder (MERN Stack)

-> This project is currently under development.

An AI-powered resume builder web application developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). The application allows users to create, edit, and generate professional resumes with AI-assisted content generation.

## Features (In Progress)
- User authentication
- Resume creation and editing
- AI-based content suggestions
- Resume preview and download
- Responsive UI

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Tailwind CSS

## Status
-> This project is still in development. New features and improvements are being added.
